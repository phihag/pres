===== ipaddress
  • Rechnen mit IP-Addressen und -Netzwerken
  • Anwendungszwecke: Firewall-Konfiguration, Logging etc.
  • ... und Lösen von Übungsaufgaben
  • Vormals ipaddr (von google)
  • In der stdlib seit Python 3.3
  • Als Paket für Python 2.6 und 2.7, pip install ipaddress
===== import ipaddress >>> import ipaddress >>> a = ipaddress.ip_address(u'127.0.0.1') >>> a IPv4Address('127.0.0.1') >>> a.version 4   # IPv4 >>> int(a) 2130706433 >>> a.packed b'\x7f\x00\x00\x01' ===== Ist das eine IP-Adresse? >>> from ipaddress import ip_address >>> ip_address(u'0:0::00') IPv6Address('::') >>> ip_address(u'::1:2:3::4') ValueError: '::1:2:3::4' does not appear to be an IPv4 or IPv6 address >>> ip_address(u'::255.0.127.0') IPv6Address('::ff00:7f00') ===== Was für eine IP-Addresse ist das? def print_ip_props(s): a = ipaddress.ip_address(s) props = (p for p in dir(a) if p.startswith('is_')) for p in props: if getattr(a, p): print(p) ===== Was für eine IP-Addresse ist das? >>> print_ip_props(u'127.1.2.3') is_loopback >>> print_ip_props(u'192.168.2.3') is_private >>> print_ip_props(u'ff5e::') is_multicast >>> print_ip_props(u'::') is_reserved is_unspecified ===== Netzwerke >>> n = ipaddress.ip_network(u'1.2.18.0/23') >>> n.num_addresses 512 >>> ip_address('1.2.19.39') in n True >>> # Netzwerk aufsplitten >>> sub = ipaddress.ip_network(u'1.2.18.0/25') >>> list(n.address_exclude(sub)) [IPv4Network('1.2.19.0/24'), IPv4Network('1.2.18.128/25')] ==== Fragen?
